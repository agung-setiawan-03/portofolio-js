About Me

Haloo, Saya adalah

Agung Setiawan, Saat ini saya adalah mahasiswa semester 5 Universitas Gunadarma jurusan Informatika Sistem. Untuk hal-hal yang membawa saya terjun ke bidang web development karena saya memiliki sesuatu yang saya sukai dari saya jurusan yaitu bagian pengembangan web membuat saya bersemangat untuk mendalami berbagai bahasa pemrograman seperti HTML, CSS, JavaScript dan bahkan bermain dalam framework seperti Bootstrap 5, TailwindCSS, ReactJS. Untuk mempelajari lebih lanjut tentang pengembangan web, saya sering mencoba membuat beberapa proyek kecil, dari situ saya mendapatkan proyek baru pengalaman yang lebih keren dari sebelumnya dan dia bercita-cita untuk mengejar karir sebagai Rekayasa Perangkat Lunak di masa depan
